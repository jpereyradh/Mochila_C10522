¿quien inventó el sistema de control de version Git y por qué?
El sistema de control de versiones fue creado por Linus Trovals para facilitar el desarrollo de Linux,
pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones
 cuando estás tienen un gran número de archivos de código fuente.

¿A quien pertenece actualmente GitHub y por qué?
GitHub fue obtenido por Microsoft por 7.500 millones de dólares.
Fue comprado porque la compañía adoptó una estrategia nueva,
mucho más abierta al mundo del software y a la integración con otras plataformas y servicios.

¿Hay otra forma que no sea la terminal para trabajar con GitHub?
Sí, se pueden usar clientes GIT con GUI (GitHub Desktop, SourceTree o Git Kraken).
