a. ¿Quién inventó el sistema de control de versión Git y por qué?

En abril del 2005 Linus Torvalds, el padre del sistema operativo Linux, diseñó un motor de control de código llamado Git con el propósito de tener un sistema 
descentralizado que le permitiera trabajar de la forma que las herramientas del momento no le permitían.
Su éxito fue inmediato y rápidamente creció su adopción por la comunidad de desarrolladores hasta convertirse en el gestor de código más utilizado en todo el mundo.




b. ¿A quién pertenece actualmente Github y por qué?

El 4 de junio de 2018, Microsoft compró GitHub por la cantidad de 7500 millones de dólares.
Según el comunicado, Microsoft consideró a GitHub como la plataforma de desarrollo de software líder en todo el mundo
