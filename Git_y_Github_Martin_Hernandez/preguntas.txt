a. ¿Quién inventó el sistema de control de versión Git y por qué?

Lo desarrolló Linus Torvalds
Git se creo para tener un control de versiones más eficiente, facil de usar y de poder usarlo en cualquier lugar sin tener que usar un pendrik, con esto se logro que este siempre sincronizado y que cualquier persona del equipo de trabajo tenga la ultima modificacion 


b. ¿A quién pertenece actualmente Github y por qué?

Atualmente es de Microsoft
Porque la empresa cambio la estrategia y busca ser más abierta al mundo del software libre y a la integración con otras plataformas y servicios, ademas porque busca competir con AWS
