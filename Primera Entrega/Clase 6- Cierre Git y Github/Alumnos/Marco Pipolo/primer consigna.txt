SEGUNDA CONSIGNA

Git init = Inicializamos un repositorio local.
git status= Chequeamos el estado de los archivos y directorios dentro del repo local.
git add  <nombredelarchivo> = Agrego el archivos o directorio con sus cambios correspondiente.
git add . = Agrega todos los archivos o directorios en estado untrucked con sus cambios correspondientes
git commit -m “mensaje” = Confirmó los cambios establecidos anteriormente y dejó registro de los mismos.
git push = Subo los cambios al repositorio remoto
git pull = Bajo al repositorio local los cambios del repositorio remoto.
