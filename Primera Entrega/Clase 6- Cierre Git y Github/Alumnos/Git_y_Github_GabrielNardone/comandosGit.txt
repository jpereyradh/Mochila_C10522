
Inicializa un repositorio local para prepararlo para la conexión a un repositorio remoto .
git init

Se puede utilizar para configurar una conexión entre un repositorio local y un repositorio remoto 
git remote add "URL del repositorio"

Crea un repositorio local haciendo una copia de un repositorio remoto en la URL especificada. Este comando también crea una rama de seguimiento remoto para cada rama del clonado.
git clone remote-url local


Muestra el alias y la URL que el repositorio local utiliza para recopilar y enviar información al repositorio remoto .
git remote -v

Envía las confirmaciones finalizadas del repositorio local al repositorio remoto.
git push origin "nombre de la rama"

Especifica el estado de cosas
git status

Añade archivos específicos a la confirmación pendiente en el repositorio local.
git add nombre

Añade todos los archivos nuevos, modificados y eliminados a la confirmación pendiente en el repositorio local.
git add .

Finaliza la confirmación pendiente en el repositorio local.
git commit -m "comentario relevante"

Enumera todas las ramificaciones del repositorio local con un asterisco (*) junto a la ramificación actual.
git branch

Muestra todas las ramificaciones del repositorio local y todas las ramificaciones de seguimiento remotas del repositorio local.
git branch -a

Crea una nueva ramificación en el repositorio local con el nombre de la ramificación especificada.
git branch nombre

Cambia a otra ramificación del repositorio local con el nombre de la ramificación especificada.
git checkout nombre

Empuja una nueva sucursal desde el repositorio local al repositorio remoto con el alias especificado de la branch
git push nombre de la branch

Combina los cambios de otra ramificación del repositorio local con la ramificación actual del repositorio local.
git merge nombre de la otra branch
