1)¿Quién inventó git y por qué?

Git es un proyecto de código abierto maduro y con un mantenimiento activo que desarrolló originalmente Linus Torvalds, el famoso creador del kernel del sistema operativo Linux, en 2005. Git se ha ideado para posibilitar la ramificación y el etiquetado como procesos de primera importancia (a diferencia de SVN) y las operaciones que afectan a las ramas y las etiquetas (como la fusión o la reversión) también se almacenan en el historial de cambios.

2)¿A quién pertenece actualmente Github y por qué?

Git pertenece actualmente a Microsoft, quien el 4 de junio de 2018, compró GitHub por la cantidad de 7500 millones de dólares.

3)¿Hay otra forma que no sea la terminal para trabajar con Github?

Github, además de trabajar por la terminal, tiene una versión desktop con un GUI más amigable, aunque en general los desarrolladores se sienten más cómodos usando la terminal.
