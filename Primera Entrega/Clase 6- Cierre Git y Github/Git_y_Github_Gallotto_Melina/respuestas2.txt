COMANDOS +UTILIZADOS HASTA EL MOMENTO
$ git init (Inicia el repositorio local)
$ git config user.name 'mi-usuario' (Agrega mi identidad)
$ git config user.email 'miCorreo@email.com' (Agrega mi identidad)
$ git config --global user.name 'mi-usuario'(Configuración permanente)
$ git config ---global user.email 'meli.gallotto@gmail.com' (Configuración permanente)
$ git add . (Agrega TODOS los archivos)
$ git commit -m 'mensaje' (Confirma los archivos agregados al paquete o commit)
$ git push origin main (Sube los archivos del repo local a Github)
$ git pull origin main (Actualiza los cambios del repositorio remoto a la computadora)
$ git branch nombreRama (Crea una rama)
$ git clone (Crea una copia exacta de un repositorio remoto en la computadora)
$ git log (Muestra el registro de commits)
$ git --version (Muestra la versión de Git)
$ git status (Muestra el seguimiento del estado de los archivos)
$ git remote add origin http://....... (Apunta el repo local al repositorio remoto)
$ git remote -v (Con este comando chequeamos que la vinculación local/remoto esté OK)
$ git push origin nombreRama (Push a una rama específica)
$ git pull origin nombreRama (Actualización de rama)
