Comandos de la terminal.

clear // limpiar consola
mkdir nombre de carpeta / crea carpeta
cd / entrar en la carpeta
cd .. / subir a la carpeta anterior
ls /ver lo que hay dentro de la carpeta
rm nombre / borra archivo
rm -r nombre /borrar directorio y todo los archivos que tiene

Comandos de Git.
git help / comando de ayuda
git config --global user.name "nombre de usuario"
git config --global user.email "email"
git init / iniciar repositorio local
git status / estado de los archivos
git add . / agrega todo los archivos
git commit - m "nombre"
git rm archivo / elimina
git log / muestra historial
git remote / repositorio externo
git push / agrega la informacion al repo 
git pull / actualizar los datos
git clone / clonar repositorio
git branch nombre / nueva rama
git rev-parse --show-toplevel
